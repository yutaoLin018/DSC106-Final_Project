Title: The Greening Earth: Mapping Global Vegetation Change from MODIS 

Main idea: This project uses NASA MODIS NDVI/EVI vegetation data to show where vegetation has increased or declined across the world over time. Inspired by the 3D map style from The Pudding(https://pudding.cool/2018/10/city_3d/), the visualization will use vertical bars or spikes on a world map to represent vegetation intensity or change. Users can compare an earlier baseline year, such as 2012, with a recent year, such as 2023, to see patterns of greening, vegetation loss, drought stress, and possible deforestation. Instead of asking “where do people live?”, this project asks: where is the Earth becoming greener, and where is vegetation disappearing?

Data set using: NASA MODIS (MODerate Resolution Imaging Spectroradiometer) Satellite Imagery https://worldview.earthdata.nasa.gov/
